# Pomodoro
pomodoro app with GUI
<br>
<br>
work session: WORK_MIN = 25
<br>
short break: SHORT_BREAK_MIN = 5
<br>
every forth break: LONG_BREAK_MIN = 20
<br>
check marks appear under the clock every finished session to show your progress
